# PiTrap Architecture

PiTrap is designed as a modular cybersecurity deception node.

Main layers:

Network Layer
Captures incoming traffic using packet sniffers.

Honeypot Layer
Simulates vulnerable services to attract attackers.

Data Collection Layer
Logs attacker commands and network traffic.

Threat Intelligence Layer
Analyzes attack patterns and generates insights.

Visualization Layer
Displays attacks through a dashboard interface.
